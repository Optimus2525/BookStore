Bookstore is a simple CRUD application created using Spring Boot, Java and Thymeleaf. It works in two ways - data can be saved either in the Collection or in a MySQL database.

To switch actions, change the profile name from "default" to "MySQL" in the application.properties file. You are currently using Thymeleaf UI. Use This link to get a UI with simple JSON data
